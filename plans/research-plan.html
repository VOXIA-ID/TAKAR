Visi & Misi

Visi: Menjadi infrastruktur agregasi pengetahuan kolektif Indonesia — mengubah opini publik menjadi sinyal probabilitas yang actionable bagi media, lembaga survei, dan pengambil keputusan.

Misi: Membangun pasar prediksi yang legal, liquid, dan mudah diakses oleh seluruh lapisan masyarakat Indonesia dengan barrier entry minimum dan mekanisme yang adil.

Apa itu Prediction Market?

Prediction market adalah bursa di mana peserta membeli dan menjual kontrak atas kemungkinan suatu kejadian terjadi atau tidak. Harga kontrak mencerminkan probabilitas kolektif — jika kontrak "YES" diperdagangkan di 0.65, artinya pasar memperkirakan kemungkinan 65%.

Penelitian dari Mellers et al. (2015) membuktikan prediction market secara konsisten lebih akurat dari polling dan model statistik konvensional dalam forecasting jangka pendek.

Database Schema

PostgreSQL sebagai primary database dengan event sourcing via CosmosDB untuk audit trail lengkap setiap transaksi. Semua nilai finansial disimpan dalam sen (integer) untuk menghindari floating point.

Konvensi: Semua nilai uang dalam BIGINT satuan sen (IDR × 100). Contoh: Rp 10.000 = 1000000. Mencegah floating-point rounding error pada kalkulasi keuangan.

Mekanisme LMSR

TAKAR menggunakan Logarithmic Market Scoring Rule (LMSR) — mekanisme AMM yang terbukti efisien untuk prediction market tanpa memerlukan counterparty langsung.

Cost Function

Cost Function — LMSR
C(qyes, qno)  =  b  ×  ln( e(qyes/b) + e(qno/b) )
Dimana b adalah parameter likuiditas. qyes dan qno adalah jumlah shares outstanding untuk masing-masing outcome.

Harga Probabilitas

Implied Probability — YES
P(YES)  =  e(qyes/b) / ( e(qyes/b) + e(qno/b) )
P(YES) selalu di rentang (0,1) dan P(YES) + P(NO) = 1 secara otomatis. Ini adalah softmax dari log-quantities.

Biaya Pembelian

Trade Cost — Beli Δq Shares YES
cost  =  C(qyes + Δq,  qno)  −  C(qyes,  qno)
Biaya adalah selisih cost function sebelum dan sesudah trade. Semakin besar Δq dibandingkan b, semakin tinggi price impact.

Implementasi TypeScript

TypeScriptlmsr.ts — Market Engine
export class LMSREngine {
  private b: number;

  constructor(liquidityB: bigint) {
    this.b = Number(liquidityB) / 100;
  }

  cost(qYes: number, qNo: number): number {
    return this.b * Math.log(
      Math.exp(qYes / this.b) + Math.exp(qNo / this.b)
    );
  }

  probability(qYes: number, qNo: number): number {
    const expYes = Math.exp(qYes / this.b);
    const expNo  = Math.exp(qNo  / this.b);
    return expYes / (expYes + expNo);
  }

  tradeCost(qYes: number, qNo: number, outcome: 'YES' | 'NO', delta: number): bigint {
    const before = this.cost(qYes, qNo);
    const after  = outcome === 'YES'
      ? this.cost(qYes + delta, qNo)
      : this.cost(qYes, qNo + delta);
    return BigInt(Math.round((after - before) * 100));
  }
}

Framing

Skill vs. Chance

Sandbox

Konten

Market Curation

Kurasi ketat jenis market. Hindari konten sensitif SARA. Fokus pada verifiable public events.
