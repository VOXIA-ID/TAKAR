# 🎯 TAKAR

> **Pasar Prediksi Indonesia** — Polymarket versi lokal, dibangun untuk konteks regulasi, budaya, dan pasar Indonesia.

---

## Apa itu TAKAR?

TAKAR adalah platform prediksi berbasis pasar pertama di Indonesia. Pengguna memasang posisi pada hasil kejadian nyata — politik, ekonomi, olahraga, budaya pop. Harga pasar mencerminkan probabilitas kolektif.

Inspirasi: [Polymarket](https://polymarket.com), [Metaculus](https://metaculus.com), [Manifold Markets](https://manifold.markets)
